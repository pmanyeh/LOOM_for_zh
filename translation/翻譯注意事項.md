# 《LOOM》繁體中文化 - 翻譯注意事項

這份文件是專門為《LOOM》中文化翻譯人員準備的指南。由於我們使用了「字型替換」的底層黑科技，在翻譯時有一些必須嚴格遵守的技術限制。請在開始翻譯前詳讀本文件。

## 1. 基本作業流程

所有的翻譯作業都在 `loom_strings.csv` 中進行：
*   **English 欄位**：原本的英文對話，請**絕對不要修改**此欄位的任何內容。
*   **Chinese 欄位**：請在此處填入您的繁體中文翻譯。如果這個欄位保持空白，遊戲中就會顯示原本的英文。

## 2. ⚠️ 嚴格的「字數長度限制」 (最重要)

因為遊戲檔案的指標無法任意移動，我們的工具被設計為「原地覆蓋」。這代表：
**【翻譯後的中文所佔用的字節大小，絕對不能超過原本的英文】**

*   **字節計算規則**：
    *   1 個英文字母 / 標點符號 / 空白 = **1 字節**
    *   1 個中文字 / 全形標點符號 = **2 字節**

**✅ 安全範例 (中文小於或等於英文)：**
*   `Skill level` (11 字節) ➡️ `難度` (4 字節) (安全！程式會自動用空白幫您補齊剩下的 7 字節，並且會自動讓文字**置中對齊**)。
*   `Please choose your skill level.` (31 字節) ➡️ `請選擇遊戲難度。` (16 字節) (安全！)

**❌ 爆炸範例 (中文大於英文)：**
*   `OK` (2 字節) ➡️ `確定` (4 字節) **(會導致遊戲崩潰！)**
    *   *解決方案*：請縮減翻譯，改為 `好` (2 字節)。
*   `Y` (1 字節) ➡️ `是` (2 字節) **(會導致遊戲崩潰！)**
    *   *解決方案*：這種單一字母無法翻譯成中文，請直接保留 `Y`。

如果您不小心寫了太長的翻譯，在執行編譯時，終端機畫面會跳出警告：
`Warning: Line XX: Translated string is longer than original (XX > XX). This might corrupt DISK01.LEC!`
**看到這個警告時，請務必回去修改縮短那句翻譯，否則遊戲執行時會發生 `Cannot read resource!` 崩潰錯誤。**

## 3. 控制代碼保留

在 CSV 檔的英文句子中，您有時會看到類似 `\255\004\041` 或者 `\015` 這種由反斜線開頭的代碼。
*   這是 SCUMM 引擎用來控制「文字顏色」、「換行」或「播放語音」的程式碼。
*   **請將這些代碼原封不動地複製到您的中文翻譯中，並放在相同的位置。**
*   千萬不要刪除、修改裡面的數字，也不要把全形字元（如 `＼`）用在這些代碼上。

## 4. 遇到「缺字」怎麼辦？

我們的腳本是「動態字庫」，目前只有產生您「已經翻譯過」的那些中文字。
當您翻譯了幾百句對話，使用到新的繁體中文字時：
1. 先存檔您的 `loom_strings.csv`。
2. 開啟終端機，執行一次 `python generate_fnt.py`。
3. 腳本會自動掃描您的 CSV 檔，把那些「新出現的字」畫成像素並擴充到 `chinese_gb16x12.fnt` 與 `charmap.json` 中。

## 5. 完整更新指令 (翻譯完畢後的驗證)

每次完成一個段落的翻譯，想要進遊戲看成果時，請依序執行以下步驟：

1. **擴充字庫 (確保沒有缺字)**
   ```powershell
   python generate_fnt.py
   ```
2. **編譯翻譯檔 (將文字轉換為神祕代碼)**
   ```powershell
   python helper.py compile loom_strings.csv loom_strings_h.txt loom_strings_translated.txt
   ```
   > ⚠️ **【極度重要警告】**：中間的樣板檔案名稱 **絕對必須是 `loom_strings_h.txt`**（有 `_h` 的版本）！
   > 千萬不要打成 `loom_strings.txt`！因為只有 `_h` 的版本才包含 `scummtr` 注入需要的 `[064:LS#0200]` 等標籤。如果打錯檔名，注入工具會因為找不到標籤而完全略過，導致遊戲裡還是英文！
   *(請在此步驟留意終端機是否有跳出 Translated string is longer than original 的警告！)*
3. **注入遊戲檔案**
   ```powershell
   ..\tools\scummtr-0.5.1-win32\scummtr.exe -w -g loomcd -h -i -f loom_strings_translated.txt -p "..\from steam\LOOM"
   ```
4. **開啟遊戲！**
   從 ScummVM 選擇 `Loom (Traditional Chinese Patch)` 即可看到最新的翻譯！
